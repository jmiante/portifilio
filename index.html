Objetivo

Este site foi criado como parte de um projeto acadêmico no curso de Ciência de Dados da faculdade UNINTER, com o objetivo de servir como um Portfólio online para destacar minhas habilidades e incluir alguns projetos desenvolvidos

Este espaço representa não apenas meu trabalho acadêmico, mas também meu crescimento pessoal e profissional neste domínio fascinante.

Ao navegar por este Portfólio, você encontrará diversas seções, sendo elas:

  • Início: Está sessão é a aba inicial do Portfólio, ou seja, a página atual.
  • Sobre: Nesta sessão será apresentado um pouco sobre mim.
  • Formações/Habilidades: Nesta irá apresentar minhas formações e minhas habilidades.
  • Portfólio: Nesta sessão será apresentado os projetos em que trabalhei (incluindo projetos criados antes do curso).
  • Contato: Nesta sessão será apresentado um modelo de formulário criado apenas com HTML/CSS (Onde o Formulário não será funcional, apenas será desenvolvido o front-end).
Requisitos

Este site foi projetado para atenter todos os requisitos solicitado para o trabalho acadêmico, onde conta com os seguintos tópicos

  • Linguagens:
    • HTML
    • CSS
    • Javascript (Opcional)
  • Páginas
    • Sobre mim: que fale sobre os hobbies, informaçoes, etc.
    • Sobre Formações: Formação Educacional, Idiomas, Habilidades, etc.
    • Portfólio: Trabalhos realizados, tanto acadêmico quanto outros
    • Contato: Fazer a parte visual do Formulário
  • Restrições
    • Não pode Utilizar Bootstrap ou Similares (Para Estilização deve utilizar o CSS)
    • Explicar o Código através de comentários
    • Não Utilizar WIX, WordPress e Framework do tipo
    • O Site deve ser desenvolvido com Linguagem pura (HTML/CSS)
    • Pode ser utilizado Javascript, porém não é obrigatório
    • Utilizar o nome como parte do dominio no servidor gratuito
